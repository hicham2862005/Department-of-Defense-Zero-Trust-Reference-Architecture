# Chapter 2: ZTA Capabilities
## Definitions:
this chapter contines **two essential terms** that **defines the system** and **it's mission**
- **Capability:** refers to the **Outcome the system must achive**, this includes: **Secure Access**, **Encryption of sensitive data** ...
- **Sub-Capability:** a **smaller piece** that **Supports to Main Capability**, such as **Identiy Providers** which supports the **Auth Capability**
